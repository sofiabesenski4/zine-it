# Graphql

## Export from Python

https://docs.graphene-python.org/projects/django/en/latest/introspection/#graphql-sdl-representation

To Regenerate the `schema.graphql` file, run:

```
python manage.py graphql_schema
```

## Import into javascript-land

```
cd frontend
npm run gql
```
